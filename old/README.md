# wcmush.com website

Currently built using:

- Ruby 1.9.2p290
- frank 1.0.9 (https://github.com/blahed/frank)

To get running:

1. gem install frank
2. git clone git@github.com:tkrajcar/wcnh_web.git (new dir)
3. cd (new dir)
4. frank server
5. http://localhost:3601/

To export to static assets:

1. frank export (target dir)
